# twi-gb
A one-person project to write a Nintendo Game Boy emulator entirely in C.

Note that, at this time, twi-gb is not yet a functional Game Boy emulator.

Milestone checklist:
- [x] OpenGL rendering integration
- [ ] CPU interpreter
- [ ] Shader-based fragment color evaluation from emulated GB VRAM.
- [ ] Input support
- [ ] Full interrupt support
- [ ] Audio support
- [ ] ...
